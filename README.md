# 🏗️ StockMaster

**StockMaster** es una aplicación de escritorio desarrollada en Unity para la gestión de stock de materiales en obras. Pensada para pequeñas y medianas empresas del sector de la construcción, permite llevar un control de entradas, salidas, inventario y trazabilidad.

---

## 🧩 Características

- ✅ Login por DNI sin contraseña
- ➕ Añadir nuevas entradas de material
- 📝 Editar y eliminar entradas
- 🧹 Interfaz limpia y sencilla con scroll y confirmaciones
- 🧍 Nombre de usuario visible tras login
- 📁 Datos persistentes con base de datos local SQLite
- Bases de datos portable 
- EN CONSTRUCCION!!!!!!

## 💻 Tecnologías

- Unity 2022.3.6f1 (LTS)
- C#
- SQLite (mediante SQLite4Unity3d)
